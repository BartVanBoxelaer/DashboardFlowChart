Step 4: Recommended reporting tool for your needs!

SAP Analysis for Office (AfO)

Analysis for Office is the preferred reporting tool within Borealis.
It is the only supported tool for detailed data analysis.
(other reporting tools are only intended for high level view on data via Dashboards)
